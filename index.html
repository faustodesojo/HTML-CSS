SELECTORES PARA CSS

  • El * se utiliza para modificar todo el html
  • Para editar un elemento se pone el nombre del elemento, por ejemplo: H1 ; p ; img ; etc...
  • El . se utiliza para editar por clase, es decir editar un elemento especifico. Por ejemplo, si tenes dos P, pero queres editar solo uno, tenes que poner dentro del p una clase con un nombre. (>p class="nombre"<)y para editarlo en css poner .nombre
  • Para editar por ID se utiliza el #. Poniéndole a un elemento id=´´nombre´´ y en css #nombre. El id a diferencia de las clases porque es unico.
  • Para editar por pseudo-clases se utiliza :hover. Esto sirve para cambiar, por ejemplo, el color de un elemento al pasar por arriba con el mouse. En el css se pone :hover y el elemento a modificar
